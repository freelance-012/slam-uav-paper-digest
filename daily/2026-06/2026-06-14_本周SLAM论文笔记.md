# 2026-06-14 无人机 SLAM 论文阅读 · 本周复盘

> 这周（06.08 - 06.14）抓到的 SLAM / VIO / VPR / 激光定位相关新论文里，跟无人机方向相关、且算力在我可承受范围内的，挑了 11 篇周末集中过了一遍。下面按我平时看工程的关注度（能不能上机、需不需要堆算力）分了三档。

抓取 142 篇 → 硬过滤剩 38 篇 → 算力评估后保留 11 篇

## 🟢 Tier-1：CPU 实时就能跑（4 篇）

这一档是周末花最多时间读的，目标是找能直接拿到 Pico / Orin Nano 跑的方案。

### 1. Vision-only UAV State Estimation for Fast Flights (A2RL Drone Racing)
- **arXiv**：2602.01860
- **作者**：Filip Novák 等
- **方向**：🎯 紧耦合前端（纯视觉 VIO）
- **门槛**：CPU 实时，单目 RGB + IMU
- **核心 idea**：A2RL 无人机竞速赛决赛方案，主打 GNSS 拒止下高速机动状态估计
- **关键点**：不上 LiDAR、不上 NPU，单目 + IMU 撑住高速；竞速级飞机对帧率/延迟要求比一般 VIO 严苛得多
- **对工作的价值**：⭐⭐⭐⭐
- **摘要**：针对 GNSS 拒止 + 杂波场景下的高速飞行，提出了一种基于单目相机 + IMU 的纯视觉 VIO 状态估计方法，可作为竞速场景下参考基线。

### 2. Efficient and Accurate Downfacing Visual Inertial Odometry
- **arXiv**：2509.10021
- **作者**：Jonas Kühne 等
- **方向**：🎯 紧耦合 VIO
- **门槛**：CPU 实时（micro/nano UAV 优化）
- **核心 idea**：专门为微型无人机优化的下视 VIO pipeline
- **关键点**：明确写"为 micro/nano UAV 设计"，和我们的场景最贴合；算力预算抠得很紧
- **对工作的价值**：⭐⭐⭐⭐⭐
- **摘要**：提出了一种高效精确的 VIO pipeline，针对微型/纳型无人机的下视相机 + IMU 场景做了系统级优化，资源占用比通用 VINS 系小一个量级。

### 3. Adaptive Covariance EKF/UKF Hybrid for VIO (Qf-ES-EKF/UKF)
- **arXiv**：2512.17505
- **作者**：Ufuk Asil 等
- **方向**：📡 多传感器融合（IMU 鲁棒自适应）
- **门槛**：CPU 实时
- **核心 idea**：把 quaternion 误差建模和自适应协方差结合起来，做松耦合 VIO
- **关键点**：对环境扰动、IMU 退化有动态评估机制；松耦合架构工程上更容易移植
- **对工作的价值**：⭐⭐⭐⭐
- **摘要**：面向无人机的 VIO 鲁棒性问题，提出 Qf-ES-EKF/UKF 混合滤波架构，结合自适应协方差估计 IMU 可靠性。

### 4. Unscented Particle Filter for Visual-Inertial Navigation (QUPF-VIN)
- **arXiv**：2504.19318
- **作者**：Khashayar Ghanizadegan 等
- **方向**：📡 多传感器融合（滤波后端）
- **门槛**：CPU 实时（粒子滤波 + IMU + 路标）
- **核心 idea**：用 quaternion 的无迹粒子滤波做 6 DoF VIO
- **关键点**：处理 6 DoF 非线性导航很自然；粒子滤波在特征点丰富时精度优于 EKF
- **对工作的价值**：⭐⭐⭐
- **摘要**：用基于四元数的无迹粒子滤波做 6 自由度视觉惯性导航，对非线性运动学建模更自然。

## 🟡 Tier-2：需要 10-100 TOPS NPU / 边缘 GPU（5 篇）

这一档本身能上 Jetson Orin Nano/NX，值得记一下但平时不会优先复现。

### 5. PanoAir: Panoramic Visual-Inertial SLAM with Cross-Time UAV Dataset
- **arXiv**：2604.00852
- **作者**：Yiyang Wu 等
- **方向**：🎯 紧耦合 VIO + 数据集
- **门槛**：NPU/边缘 GPU（全景 + 视觉惯导）
- **核心 idea**：用全景相机做 VIO，解决窄 FoV 漂移问题，并放出一个跨时间的真实 UAV 数据集
- **关键点**：数据集是亮点，长期复现的视觉变化；全景接入对算力要求高一档
- **对工作的价值**：⭐⭐⭐⭐
- **摘要**：针对现有 VI-SLAM 在 UAV 上窄 FoV 漂移问题，提出基于全景相机的方案，并发布了一个跨时间的真实 UAV 数据集。

### 6. PiLoT: Neural Pixel-to-3D Registration for UAV Geo-localization
- **arXiv**：2603.20778
- **作者**：Xiaoya Cheng 等
- **方向**：🛰️ 视觉 + GNSS
- **门槛**：NPU（神经网络 + GPU 推理）
- **核心 idea**：把 UAV 自定位和目标定位统一到一个 neural pixel-to-3D 框架
- **关键点**：替代 GNSS + 激光测距的多模块方案；统一网络工程上更省事
- **对工作的价值**：⭐⭐⭐
- **摘要**：用统一的神经像素到 3D 配准框架同时做 UAV 自定位和目标定位，替代传统的 GNSS + 激光测距解耦方案。

### 7. PA-LVIO: Real-Time LiDAR-Visual-Inertial Odometry with Pose-Only BA
- **arXiv**：2603.16228
- **作者**：Hailiang Tang 等
- **方向**：📡 多传感器融合（LVI）
- **门槛**：边缘 GPU / Jetson Orin
- **核心 idea**：把 LVI 的 BA 从联合优化改成只优化位姿部分，实时性大幅提升
- **关键点**：号称 real-time；和 FAST-LIO2 比加了一层视觉约束；只优化 pose 不优化 map point，工程上很聪明
- **对工作的价值**：⭐⭐⭐⭐
- **摘要**：提出 PA-LVIO，使用仅位姿 BA 的紧耦合 LVI 方案，保留精度的同时大幅降低计算量。

### 8. AERO-VIS: Asynchronous Event-based Real-time Onboard VIO
- **arXiv**：2605.07885
- **作者**：R²Lab
- **方向**：📍 事件相机 + IMU
- **门槛**：NPU / 事件相机专用处理
- **核心 idea**：第一个纯事件相机的紧耦合 VIO，能闭环控制 UAV
- **关键点**：高速 + HDR 场景下的唯一解；事件相机硬件门槛和价格都在降；论文里 demo 了闭环控制，证据链完整
- **对工作的价值**：⭐⭐⭐⭐⭐
- **摘要**：首个纯事件相机的惯性 SLAM 系统，实现了 UAV 闭环控制和大范围状态估计，针对高速/低照度场景设计。

### 9. Edged USLAM: Edge-Aware Event-Based SLAM with Learning-Based Depth Priors
- **arXiv**：2603.08150
- **作者**：Şebnem Sarıözkan 等
- **方向**：📍 事件相机 + SLAM
- **门槛**：NPU（学习深度 + 事件流）
- **核心 idea**：在边缘设备上跑的事件相机 SLAM，引入学习型深度先验
- **关键点**：名字里的"Edged" = edge device；学习型深度轻量化是亮点
- **对工作的价值**：⭐⭐⭐⭐
- **摘要**：传统视觉 SLAM 在快速运动/低照度下失效，本文用事件相机 + 学习型深度先验，在边缘设备上实现稳定跟踪与建图。

## 🔴 Tier-3：算力超限 / 思路追踪（2 篇）

这一档现在不会去复现，但思路值得记。

### 10. HE-VPR: Height Estimation Enabled Aerial VPR
- **arXiv**：2603.04050
- **作者**：Mengfan He 等
- **方向**：🔁 回环 / 位置识别
- **门槛**：GPU 推理（冻结 DINOv2）
- **核心 idea**：在 VPR 之前单独预测飞行高度，再做位置识别，缓解高度变化带来的尺度差异
- **关键点**：高度估计算法解耦设计很优雅；DINOv2 主干冻结，所以推理不算"全尺寸"
- **对工作的价值**：⭐⭐⭐（思路可借鉴）
- **摘要**：把高度估计从 VPR 中解耦，与冻结的 DINOv2 共享主干，应对 UAV 视觉定位中的尺度变化问题。

### 11. Altitude-Adaptive Vision-Only Geo-Localization for UAVs
- **arXiv**：2602.23872
- **作者**：Xingyu Shao 等
- **方向**：🔁 回环 / 位置识别
- **门槛**：GPU 推理
- **核心 idea**：单目下视图像做高度自适应定位，用频域 + RAC 回归分类做高度估计
- **关键点**：单目 + 频域处理很省事；和 HE-VPR 是同一波人在做
- **对工作的价值**：⭐⭐⭐
- **摘要**：单目下视图像的频域分析 + 回归-分类(RAC)方法做高度估计，再用于 VPR 匹配。
