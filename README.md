# 无人机 SLAM 论文精读笔记

每周精读无人机 SLAM 领域的最新论文，用工程视角解读学术研究。

我不是在写论文综述，我在做的是：
- 这篇论文解决了什么**实际问题**？
- 它的方法在**机载落地**上有什么坑？
- 它对**低空经济/无人机行业**有什么意义？

---

## 内容结构

每篇笔记按算力档位分类，帮你快速判断能不能上机：

| 档位 | 含义 |
|:---|:---|
| 🟢 Tier-1 | CPU 实时，板载 NUC/TX2 级别能跑 |
| 🟡 Tier-2 | 需要 10-100 TOPS NPU / 嵌入式 GPU（Orin/Xavier）|
| 🔴 Tier-3 | 算力超限，仅做趋势跟踪 |
